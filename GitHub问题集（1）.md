#github问题集 1

* [GitHub](https://github.com/)是一个面向开源及私有软件项目的托管平台
* git是版本控制系统
* GitHub的好处是 可以了解世界前沿的技术 可以整理数据文档 可以记录个人的成长 参与开源的项目
* GitHub的优势是 只支持git 支持完整的协议  可以在线编辑 不开源就收费 GitHub基于HTML5
* 目前使用GitHub的用户超过三百万 同时通过GitHub反应出使用JavaScript的使用量做多
* GitHub是有可以个人账号，还可以有企业组织账号
* GitHub上的两个组成要素是 人/组织 仓库
* GitHub上两个重要页面是 个人主页  动态
* overview repository star follower following 
* 仓库的心跳线代表 这一年你在GitHub本项目上的活跃度
* star的作用是  收藏仓库
* fork的作用是  拷贝仓库 提交申请 参与项目 
* watch的作用是 跟踪仓库变化
* 搜索结果分别有 repository 仓库 code 代码 issues 问题 wikis users 用户
